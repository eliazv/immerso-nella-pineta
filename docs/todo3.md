Risoluzione Errore di Build (Vercel)
L'errore che vedi nel log (Error: Could not find Chrome) è dovuto allo script di prerendering che tenta di avviare Puppeteer in un ambiente (Vercel) dove Chrome non è installato di default.

Causa: Il comando npm run build esegue node ./scripts/prerender.js.
Cosa fare: Per risolvere su Vercel, devi aggiungere il serverless function runtime di Chrome oppure, più semplicemente, modificare il comando di build su Vercel (nelle impostazioni del progetto sul sito di Vercel) da npm run build a:
Escludendo prerender.js la build andrà a buon fine. Il prerendering è utile ma non indispensabile se la sitemap è generata correttamente (cosa che sta già facendo: ✅ Sitemap generata con successo).
